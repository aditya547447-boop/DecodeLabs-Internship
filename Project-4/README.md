# ☁️ Project 4 - AWS Lambda Cost Calculator

## 📌 Overview
This project demonstrates a simple serverless application using AWS Lambda. The Lambda function receives two numbers as input through a JSON event and returns their sum.

## 🚀 Features
- AWS Lambda Function
- Serverless Computing
- JSON Input Processing
- Python Runtime
- Real-time Cost Calculation (Addition)

## 🛠️ Technologies Used
- AWS Lambda
- Python 3.x
- JSON

## 📥 Input Example

```json
{
  "num1": 15,
  "num2": 25
}
```

## 📤 Output Example

```json
{
  "Sum": 40
}
```

- Deploying Serverless Applications
- Handling JSON Events
- Testing Lambda Functions
