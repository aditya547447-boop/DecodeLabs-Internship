# Project 2 - Amazon EC2 with Nginx

## Objective
Launch an Amazon EC2 instance and configure the Nginx web server.

## Tools Used
- Amazon EC2
- Amazon Linux
- Nginx

## Steps Performed
1. Launched an EC2 instance.
2. Connected to the instance using SSH.
3. Installed Nginx.
4. Started the Nginx service.
5. Verified the default Nginx web page using the EC2 Public IP.
